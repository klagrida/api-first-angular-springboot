Documentation Structure
======================

Naming Convention
=================

- ALL-CAPS-WITH-DASHES.md for guides (QUICK-START.md, SETUP-GUIDE.md)
- lowercase-with-dashes.md for articles (api-first-angular-springboot.md)
- README.md for index files

Total Documentation: ~15,000 words, ~75 minutes reading time
